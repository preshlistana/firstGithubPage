What I learned

Today I learned from CodingTrain on how to create a github page.

It works like this:

  • 1) Create a gh-pages branch
  • 2) Delete the master branch
  • 3) Create an index.html file in your repo
  • 4) Go to settings and click the provided URL (your index.html file should show up first
  • 5) TIP: You can customize your url by buying your domain
